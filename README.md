# gladGAN: Generative Adversarial Networks for Graph-Level Anomaly Detection

An adaptation of [Schlegl's et al. (2019)](https://doi.org/10.1016/j.media.2019.01.010) f-AnoGAN, originally proposed for images, to graph-level anomaly detection. For more information, see [page](https://mbaranr.github.io/projects/gladgan/).

## Installing Dependencies

```
$ conda env create -f environment.yml
$ conda activate gladgan
```
